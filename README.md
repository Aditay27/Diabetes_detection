# Diabetes_detection
This project predicts whether a person is diabetic or not depending on several factors like no. of pregnancies,age,blood pressure,insulin level,glucose level etc. Kaggle dataset is used for this project.Python Streamlit framework is used to make the web app and ML algorithm used is Support Vector Machine.This model gives 77% accuracy.
